## 과제 회고

이번 과제를 수행하면서 Python을 활용한 데이터 분석의 기본 흐름을 전체적으로 공부할 수 있었다. NumPy 과제에서는 배열의 차원, shape, 데이터 타입, 인덱싱, 연산, 유니버셜 함수 등을 다루며 데이터가 어떤 구조로 저장되고 계산되는지 이해할 수 있었다. 특히 `reshape`, `ndim`, `dtype`, 인덱싱 등을 직접 사용해보면서 배열의 형태를 정확히 파악하는 것이 이후 데이터 분석의 기초가 된다는 점을 느꼈다.

Pandas 과제에서는 Series와 DataFrame을 생성하고, 필터링, 그룹화, 병합, 결측치 처리, 피벗, 중복 제거, 문자열 처리 등을 실습했다. 처음에는 `groupby()`, `merge()`, `pivot_table()`처럼 옵션이 많은 함수들이 헷갈렸지만, 여러 문제를 반복해서 풀면서 어떤 상황에서 어떤 함수를 사용해야 하는지 조금씩 구분할 수 있게 되었다.

데이터 시각화 과제에서는 정형 데이터와 비정형 데이터를 다루고, 막대그래프, 히스토그램, 산점도, 박스플롯, 시계열 그래프 등 다양한 시각화 방법을 실습했다. 단순히 데이터를 표로 확인하는 것보다 그래프로 표현했을 때 데이터의 분포, 비교, 관계, 변화 추세를 훨씬 직관적으로 파악할 수 있다는 점이 인상 깊었다. 또한 그래프의 제목, 축 이름, 범례 등을 설정하는 과정이 분석 결과를 더 명확하게 전달하는 데 중요하다는 것도 느꼈다.

이번 과제를 통해 데이터 분석은 단순히 코드를 작성하는 것이 아니라, 데이터를 이해하고 정리한 뒤 적절한 방식으로 표현하는 과정이라는 것을 배웠다. 아직 함수의 세부 옵션을 완전히 익숙하게 사용하는 데에는 시간이 더 필요하지만, 반복적인 실습을 통해 기본 개념과 사용 흐름은 많이 익힐 수 있었다. 앞으로는 단순히 예제 코드를 따라 쓰는 것에서 더 나아가, 실제 데이터에 직접 적용해보며 문제 상황에 맞는 분석 방법을 선택하는 능력을 키우고 싶다.
